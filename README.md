# ホテルKa 省エネ・創エネ ダブルプラン（ナレーション付きHTML）

このフォルダは、ブラウザで開くだけで動くナレーション付き提案資料です。
`index.html` をダブルクリックすれば表示できます。GitHub Pages に置けば URL で市村様へ共有できます。

---

## 1. フォルダ構成

```
HotelKa_DoublePlan_Web/
├─ index.html        ← 本体（これを開く／GitHubに置く）
├─ README.md         ← この指示書
└─ mp3/              ← ナレーション音声を入れる場所
   ├─ narr-01.mp3
   ├─ narr-02.mp3
   ├─ … narr-08.mp3
```

ナレーションは **8本**（スライド8枚に対応）。`mp3/` に下の名前で入れるだけで自動再生されます。

## 3. MP3の作り方（mp3gen）

1. 上の8本の原稿を、1本ずつ mp3gen に入力（音声 `ja-JP-NanamiNeural`／速度 +0%）。
2. 出力ファイル名を **narr-01.mp3 〜 narr-08.mp3** にする。
3. できた8本を、この `mp3/` フォルダに入れる。
4. `index.html` を開き、右下の「▶ ナレーション」を押すと、ページ送りに合わせて自動再生されます。

（音声が無くても資料は問題なく表示されます。各ページ下部に原稿テキストが出ます。）

## 4. GitHubで市村様へ届ける手順

1. GitHub で新規リポジトリを作成（例：`hotelka-proposal`、Public）。
2. このフォルダの中身（`index.html` / `mp3/` / `README.md`）をアップロード。
3. リポジトリの **Settings → Pages** を開く。
4. **Source: Deploy from a branch** → **Branch: main / (root)** を選び Save。
5. 1〜2分後に公開URL（`https://ユーザー名.github.io/hotelka-proposal/`）が表示されます。
6. そのURLを市村様へお送りください（スマホ・PCのブラウザでそのまま閲覧可）。

## 5. 注意（数字の扱い）

- 金額・削減効果はすべて **概算** です。最終は **デマンドデータの精査・現地調査・接続検討（負担金）** のあとに確定します。
- ⑦外壁トリプルガード（566㎡×10,800円＝約611万・概算）を含むDの費用は概算です。
- 遮熱の内訳は 窓25%＋屋上5%＋外壁3%＝空調 約33%減（概算）。これにより低圧化の単価上昇分を相殺する設計です。
- ⑤空調機器交換の削減効果は **経年で年約3%逓減** します。長期（複数年・回収）試算では必ずこの逓減を反映します。
- 20年比較の通常側は **塗装4,000円/㎡・陸屋根防水6,000円/㎡**（防水相場 3,000〜10,000円/㎡の中位）＋足場で1回 約836万・+6%/年で2回。①低圧化40万・②太陽光390万（工事260＋架台130）で反映。
- 単品の削減額は「単独導入時」の目安で、**合算は単純合計になりません**（空調系の効果は重複）。
